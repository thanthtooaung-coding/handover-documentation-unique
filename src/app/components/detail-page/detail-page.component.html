ခြုံငုံသုံးသပ်ချက်

Mifos X Fineract သည် အဏုငွေချေးငွေလုပ်ငန်းများအတွက် ဖန်တီးထားသော open-source core banking platform ဖြစ်သည်။ ၎င်းသည် ငွေကြေးဆိုင်ရာ ဝန်ဆောင်မှုများကို ထိရောက်စွာ စီမံခန့်ခွဲရန်အတွက် comprehensive solution တစ်ခုဖြစ်ပြီး၊ လုပ်ငန်းများ၏ လိုအပ်ချက်များကို ဖြည့်ဆည်းပေးသည်။

ရည်မှန်းချက်များ

  • အဏုငွေချေးငွေလုပ်ငန်းများကို ဒစ်ဂျစ်တယ်စနစ်ဖြင့် စီမံခန့်ခွဲရန်
  • ဖောက်သည်များ၏ အချက်အလက်များကို ဘေးကင်းစွာ သိမ်းဆည်းရန်
  • ငွေကြေးဆိုင်ရာ လုပ်ငန်းစဉ်များကို အလိုအလျောက် လုပ်ဆောင်ရန်
  • အစီရင်ခံစာများနှင့် analytics များကို ထုတ်လုပ်ရန်

Deployment လုပ်ငန်းစဉ်

Backend Deployment (Fineract)

လုပ်ဆောင်ရမည့် အဆင့်များ:

  1. WAR file ကို /home/ubuntu directory သို့ upload လုပ်ရန်
  2. sudo su ဖြင့် root access ရယူရန်
  3. deploy_fineract.sh script ကို run လုပ်ရန်
  4. Pentaho JAR files များကို restore လုပ်ရန် move_pentaho_jar.sh run လုပ်ရန်
  5. (Optional) PRPT files များကို update လုပ်ရန် move_prpt_files.sh run လုပ်ရန်

Deployment Scripts:

# Backend deployment
sudo ./deploy_fineract.sh

# Restore Pentaho JAR files (Required after each deployment)
sudo ./move_pentaho_jar.sh

# Update PRPT files (Optional)
sudo ./move_prpt_files.sh

Frontend Deployment (Web App)

AWS Lightsail Deployment:

  1. Code ကို GitHub repository သို့ push လုပ်ရန်: unique-web-app-aws
  2. CI/CD pipeline ပြီးစီးသည်အထိ စောင့်ရန်
  3. sudo su access ဖြင့် Docker commands များ run လုပ်ရန်
# Stop and remove existing container
docker stop unique-web-app-aws && docker rm unique-web-app-aws && docker rmi dockerhubmsis/unique-web-app-aws:latest

# Run new container
docker run -d -p 4200:80 --name unique-web-app-aws --restart unless-stopped -v /unique/frontend:/app dockerhubmsis/unique-web-app-aws:latest

အရေးကြီးသော မှတ်ချက်များ:

  • Production deployment မလုပ်မီ staging environment တွင် အရင် test လုပ်ရမည်
  • Database backup လုပ်ပြီးမှ deployment လုပ်ရမည်
  • Deployment အပြီးတွင် system health check လုပ်ရမည်
